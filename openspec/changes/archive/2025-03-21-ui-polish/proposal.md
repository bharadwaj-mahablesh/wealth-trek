## Why

The current UI is functional but basic. As this will be hosted as a self-service website for individuals to track their net worth periodically, the look and feel needs to be professional, polished, and trustworthy. Users are entering sensitive financial data, so the interface should inspire confidence while remaining clean and not flashy.

## What Changes

- Redesign the overall layout with better visual hierarchy and spacing
- Add a refined color palette with subtle gradients and professional tones
- Improve typography with better font weights and sizes for readability
- Add subtle animations and transitions for a polished feel
- Enhance card designs with better shadows, borders, and hover states
- Improve form styling with better input focus states and validation feedback
- Add visual indicators for progress and status (loading states, success feedback)
- Improve the header with a more distinctive brand presence
- Add empty states with helpful guidance for new users
- Enhance the net worth summary with better data visualization
- Improve mobile responsiveness and touch interactions
